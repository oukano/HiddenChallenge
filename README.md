## installation

1. $ git pull https://github.com/oukano/HiddenChallenge.git 
2. $ composer install 
3. $ npm install 
4. import DB by inserting hidden.sql file found in the repo 
5. $ php artisan migrate 
6. $ php artisan serve 

## Techs 

1. Laravel 5.7 
2. VueJs 2 
3. Axios
